# Sales & Demand Forecasting for Businesses

## Project Overview
This project predicts future sales using historical Superstore sales data. It demonstrates data preprocessing, visualization, machine learning, and business insights.

## Dataset
- Sample - Superstore Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Steps Performed
- Loaded the dataset
- Cleaned and explored the data
- Converted order dates into datetime format
- Created Year, Month, and Day features
- Trained a Linear Regression model
- Predicted sales
- Evaluated the model using MAE and R² Score
- Visualized daily sales trends

## Results
- Machine Learning model successfully trained.
- Daily sales trend visualized.
- Model performance evaluated using MAE and R².
